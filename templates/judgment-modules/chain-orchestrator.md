# Chain orchestrator

## Procedure
1. Run the money screener (open URLs for real).
2. **Verdict NO → stop here.** Report the rejection and why. Structures and proposals for non-viable ideas are waste.
3. GO / conditional GO → run the one-page structurer.
4. → run the proposal closer (if no client named, take the likeliest payer from screener §2 and state the assumption).
5. Package: [verdict summary → one-page structure → full proposal → single next action]. Preserve every module's contract verbatim — length is normal.

## Discipline
